Política de Privacidade

1. Informações Coletadas

Coletamos apenas informações necessárias para operar a integração com o Notion:

  • Token de acesso OAuth do Notion
  • ID do usuário e workspace
  • Informações de páginas acessadas através da API

2. Uso das Informações

Usamos os dados apenas para:

  • Autenticação com a API do Notion
  • Operação dos recursos da integração
  • Melhorar a experiência do usuário

3. Armazenamento

Os dados são armazenados de forma segura e não são compartilhados com terceiros.
